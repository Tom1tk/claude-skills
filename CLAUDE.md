# Global Claude Instructions

## Code Style
- Prefer explicit over clever
- Always add error handling
- Write self-documenting code; comments explain *why*, not *what*

## Behaviour
- Be concise. Skip preamble.
- When uncertain, ask one clarifying question before proceeding
- Prefer editing existing files over creating new ones unless asked

## Stack Awareness
- Primary languages: Python, TypeScript, Bash
- Infra: Proxmox, Docker/LXC containers, Linux
